# Monitor Content Setup

The monitor screen can display either static images or animated videos.

## Current Setup:
- **Image**: `monitor-wallpaper.jpg` (currently a gaming setup wallpaper)
- **Video**: `monitor-video.mp4` (optional - not currently present)

## How to Change Monitor Content:

### For Static Images:
1. Replace `monitor-wallpaper.jpg` with your desired image
2. Recommended resolution: 1920x1080 or 16:9 aspect ratio
3. Supported formats: JPG, PNG

### For Videos:
1. Add an MP4 file named `monitor-video.mp4` to this directory
2. The code will automatically prefer video over static images
3. Recommended specs:
   - Resolution: 1920x1080
   - Format: MP4 (H.264)
   - Duration: Short loops work best (5-30 seconds)
   - Keep file size reasonable (under 50MB)

### Using the Download Script:
Run `./download-monitor-content.sh` to download pre-selected wallpapers:
1. Gaming desktop wallpaper
2. Cyberpunk desktop wallpaper  
3. Matrix/code style wallpaper
4. Futuristic UI wallpaper
